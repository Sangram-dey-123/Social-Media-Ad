📊 Ad Events Analysis using GMM and LSTM
📌 Project Description

This project focuses on analyzing advertisement event data using machine learning and deep learning models. The main objective is to compare the performance of:

Gaussian Mixture Model (GMM)
Long Short-Term Memory (LSTM)

The project evaluates how well these models can understand patterns in ad data and predict outcomes.

🎯 Objectives
Analyze ad event dataset
Apply GMM for clustering and pattern recognition
Apply LSTM for sequential data prediction
Compare model performance using accuracy
Visualize results using graphs
🧠 Models Used
🔹 GMM (Gaussian Mixture Model)
Used for clustering data
Helps identify hidden patterns
Works on probability distribution
🔹 LSTM (Long Short-Term Memory)
Deep learning model
Handles sequential/time-series data
Gives better accuracy for event-based predictions
📊 Results
LSTM achieved higher accuracy compared to GMM
GMM is useful for understanding data distribution
Visualization clearly shows performance comparison
